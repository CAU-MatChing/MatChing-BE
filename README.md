# 맛칭 server  
## ✨2022 중앙대학교 다빈치 SW·AI 공모전 최우수상 수상✨  
'맛칭(MatChing)'은 중앙대학교 학생들을 위해 학교 근처의 맛집을 중심으로 식사 약속 매칭 서비스를 제공하는 웹 플랫폼입니다.  
  
📍 개발 프레임워크: Django  
📍 서버 배포: AWS EC2
